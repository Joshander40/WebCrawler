To run the program simple click the run button

To determine how many website you wish to crawl change what x must be greater then on line 115

After the crawler is done crawling the GUI will open and display the gathered links

You can then click on any link to open the website or enter a search term to have the crawler crawl the gather sites and display each sites rank

When you are done using the program click the X at the top right to terminate it


If you want to create new databases use create_database() and create_rank_database() respectivly in the database.py file at the bottom
